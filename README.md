# 🌑 VESPER

> **The quiet absolute.** 
> _Forged in the shadows of forgotten altitudes. This is not a morning drink. It is a midnight ritual._

**Vesper** is a conceptual, hyper-premium coffee brand website. Designed with a strict focus on brutalism, dark mode aesthetics, and cinematic storytelling, this project prioritizes atmosphere over density. 

---

## ✨ The Aesthetic

We discarded the noise. We killed the light. What remains is absolute. 

- **Atmosphere over density**: Sweeping negative space and minimalist copy.
- **Sensory Interaction**: Elements emerge from the shadows via scroll-linked animations and respond to the cursor with magnetic pulls.
- **Responsive Parallaxing**: Layered parallax effects that gracefully adapt across desktop and mobile viewing experiences.

---

## 🛠️ Technical Architecture

Built for flawless performance and immersive scrolling:

* ⚡ **Framework**: [Next.js (App Router)](https://nextjs.org/)
* 🎨 **Styling**: Tailwind CSS & plain CSS (optimized for hardware-accelerated transforms)
* 🎬 **Animations**: [Framer Motion](https://www.framer.com/motion/) powering fluid page transitions and sequential storytelling
* 🖋️ **Typography**: Custom-configured Google Fonts (`Cormorant Garamond` for display, `Space Grotesk` for utility) via `next/font`

---

## 📂 Architecture

The project is structured purely around distinct conceptual acts:

* 🎭 **`Hero.tsx`** — The dark-mode entrance.
* ☕ **`Products.tsx`** — The canonical lineup featuring interactive vertical image parallax.
* 📜 **`Story.tsx`** — The origin paragraph revealed dynamically on scroll.
* 🧲 **`CallToAction.tsx`** — The final initiation into the Inner Circle, locked behind a magnetic button pattern.

---

## 🚀 Getting Started

Begin the ritual on your local machine:

**1. Install dependencies**
```bash
npm install
```

**2. Awaken the server**
```bash
npm run dev
```

**3. Experience Vesper**
Open [http://localhost:3000](http://localhost:3000) in your browser.
